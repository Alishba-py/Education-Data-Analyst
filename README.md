## 📌 Project Overview

This project analyzes **100,000 records** of Nigerian school admission data to uncover patterns in applications, acceptances, and enrollments over 11 academic years. A **Linear Regression model** is then used to forecast future enrollment up to 2030.

### 🔑 Key Findings

| Finding | Detail |
|--------|--------|
| 📈 Consistent Growth | Enrollment grew from **1.1M (2014)** to **6.7M (2024)** — a 6× increase |
| ✅ Stable Acceptance Rate | Steady at **~67.5%** every year |
| 🎯 High Conversion Rate | **~83%** of accepted students actually enroll |
| 📉 Post-2019 Plateau | Growth slowed to near **0%** after 2019 — possible capacity limit |
| 🔮 2030 Forecast | Model predicts **~11.1 Million** enrolled students by 2030 |
| 📊 Strong ML Model | **R² = 0.99** — extremely accurate linear regression fit |

##  Dataset

- **Source:** [Hugging Face — electricsheepafrica/nigeria-education-admission-trends](https://huggingface.co/datasets/electricsheepafrica/nigeria-education-admission-trends)
- **Size:** 100,000 rows × 9 columns
- **Years:** 2014 – 2024
- **Country:** Nigeria

### Columns

| Column | Description | Type |
|--------|-------------|------|
| `trend_id` | Unique record identifier | Text |
| `school_id` | Unique school identifier | Text |
| `academic_year` | Academic year e.g. `2016/2017` | Text |
| `num_applications` | Number of students who applied | Float |
| `num_accepted` | Number of students accepted | Float |
| `num_enrolled` | Number of students who enrolled | Float |
| `acceptance_rate` | Accepted ÷ Applied (0–1) | Float |
| `enrollment_rate` | Enrolled ÷ Accepted (0–1) | Float |
| `country` | Country name (Nigeria) | Text |

## 📈 Analysis Steps

### Step 1 — Load Data
```python
from datasets import load_dataset
ds = load_dataset("electricsheepafrica/nigeria-education-admission-trends")
df = ds["train"].to_pandas()
```

### Step 2 — Explore & Clean
- Checked shape: **100,000 rows × 9 columns**
- Found ~2% missing values in numeric columns
- Fixed with **Median fill** (numbers) and **Forward fill** (country)

### Step 3 — Feature Engineering
```python
# Extract year from academic_year string
df["year"] = df["academic_year"].str[:4].astype(int)

# Group by year
yearly_data = df.groupby("year")[["num_applications","num_accepted","num_enrolled"]].sum().reset_index()

# Growth rate & moving average
yearly_data["growth_rate"] = yearly_data["num_enrolled"].pct_change() * 100
yearly_data["moving_avg"]  = yearly_data["num_enrolled"].rolling(3).mean()
```

### Step 4 — ML Forecast
```python
from sklearn.linear_model import LinearRegression

X = yearly_data["year"].values.reshape(-1, 1)
y = yearly_data["num_enrolled"]

model = LinearRegression().fit(X, y)

future_years = np.array(range(2025, 2031)).reshape(-1, 1)
predictions  = model.predict(future_years)
```
